一句话 — AI4S 的模型 / 平台层已被深势、晶泰、巨头(MatterGen/GNoME)和开源(DeePMD)三向夹击；而它脚下的「卖铲子层」——科学数据、模拟数据、模型评测、实验中间件、科研 agent 工具链、按结果收费的验证服务——既是持续刚需、又不押单一 IP，且在中国大多还没人专门做。其中最像「中国 whitespace」的三格：① 科学专精的专家数据/RLHF(通用巨头 Surge/Mercor 不碰垂直科学)、② 实验室中间件 / lab OS + agent↔仪器协议(中国硬件强、软件层空)、③ 中立的科学模型评测即服务(MLIP Arena 类还在学术阶段)。
目录 为什么卖铲子 ① 科学数据 & 专家 RLHF ② 模拟数据 / 数据飞轮 ③ 模型评测 benchmark ④ 实验自动化中间件 / lab OS ⑤ 科研 agent 工具链 / 协议 ⑥ CRO-like 按 outcome 验证 总表 给中国创业者

为什么是「卖铲子」

淘金热里最稳的生意是卖铲子和牛仔裤。AI4S 同理:

  • 底座被夹击：材料/分子大模型(势函数、生成)正被 GNoME、MatterGen、DeePMD/DPA 等开源 + 大厂免费拉平,纯做模型窗口在关。
  • 铲子是持续刚需：所有做 AI4S 的人(包括深势/晶泰/巨头/药企)都要买数据、评模型、连仪器、跑验证——这层需求与谁赢无关、复购稳。
  • 不押单一 IP、毛利可控：避开「赌一个材料/一款药」的重资产与长周期,用工具/服务/数据现金流活着。

下面把这层拆成 6 个子赛道。每个标国外玩家 / 中国现状 / 拥挤度 / 中国切口(🟢 空白可切 / 🟡 有切口需差异化 / 🔴 已挤/巨头压)。

①高质量科学数据 & 专家 RLHF 🟢→🟡

是什么：给训练 AI4S / 科学推理模型提供「专家级」数据——材料/化学/生物博士标注、实验记录结构化、科学推理轨迹、偏好对(RLHF)、领域 eval 集。

国外：通用专家数据巨头已起飞——Surge AI(自述 $1B+ ARR、$25B 估值 🟡)、Mercor($2B→$10B 估值 / 2025 🟡)、Turing、Micro1、Snorkel,主打「科学/医学/代码专家进训练环」;数据标注市场据测 $1.2B→$10.2B(2034)🟡。但它们偏通用 LLM,不专精「材料/化学/实验」这种要懂 DFT、会读谱图、能判可合成性的硬核科学数据。

中国现状：通用数据有海天瑞声(主板上市)、数据堂、云测等 🟡;科学专精的专家数据/RLHF 基本没人单独做。

中国切口 🟢：做「科学家在环」的数据公司——签约材料/化学/药学 PhD + 把模拟(DFT/MD)与湿实验结果作为「自动标注的真值」,产出可验证的高质量科学数据/偏好对,卖给国产 AI4S 模型(深势/华为/高校)。护城河=领域专家网络 + 仿真验证飞轮,海天瑞声那套通用众包做不了。

②模拟数据集 / 数据飞轮 🟡

是什么：用 DFT/MD/第一性原理大规模算出来的结构-性质数据集,以及把它做成「datasets-as-a-service / 数据飞轮」。

国外：多为开放公共品——Materials Project、Open Catalyst(OC20/22)、Meta 的 OMat24、OMol25 等;商业化「按需算数据卖数据」反而少 🟡。

中国现状：深势/AISI 有大规模 DPA 训练语料但不单卖;公共数据平台偏学术。

中国切口 🟡：这格资本/算力密集(要堆 DFT 机时),纯卖通用数据难赚;更现实是「为某垂直行业按需生成专用模拟数据 + 配套微调」(如某类电解液/催化剂),或当第②与第①的结合体(模拟真值喂专家数据)。别正面拼「最大开放数据集」。

③科学模型 benchmark / 评测即服务 🟢

是什么：给势模型(MLIP)、材料/分子大模型做中立、可信、持续更新的评测榜与诊断——治「各家自报 SOTA、benchmark 数据泄漏、樱桃挑选」的乱象(深势的「100× / 71%」、Sakana 的「CUDA 100×」都是反面教材)。

国外：MLIP Arena(atomind-ai 在做,已是创业雏形)、Matbench Discovery 榜、LAMBench(深势/AISI, Peng 2026)、OMol25 榜、MLIPAudit——但多停在学术/开源、没人做成商业「评测即服务」 🟡。

中国切口 🟢：做「AI4S 的 MLPerf / 第三方质检」——中立评测平台 + 给药企/材料厂/基金做「这个模型/这家供应商到底行不行」的尽调报告,按报告/订阅收费。卖的是信任与中立(所以最好别自己也卖模型)。门槛在评测体系的科学严谨度与公信力。

④实验自动化中间件 / lab OS 🟢

是什么：把异构的实验仪器/机器人/液体处理站连起来、调度、记录、对接 AI 决策的中间件层——「实验室的操作系统」。这是自驱实验室真正的瓶颈(不是缺机器人,是缺把它们粘起来的软件)。

国外：Artificial.com(全实验室编排调度, arXiv 2504.00986)、Emerald Cloud Lab(远程 API 云实验室)、Strateos;开源 ChemOS / Helao / AresOS;标准 SiLA2、Bluesky、ROS 🟢。

中国现状：硬件层强(镁伽 Megarobo 实验室机器人、各院所自动化站),但「lab OS / 编排中间件」软件层基本空白;深圳已组「多 AI-多机器人」造材料 🟡,仍偏项目制非产品。

中国切口 🟢：做厂商中立的实验室中间件/OS——适配国产仪器与机器人,提供调度、数据、协议、AI 接口,卖给所有想搭自驱实验室的药企/材料厂/高校。中国制造 + 机器人成本优势在这里是顺风;难点是仪器协议碎片化(也正是壁垒)。

⑤科研 agent 工具链 / agent↔仪器协议 🟢

是什么：给「科研 agent」做基础设施——检索领域文献/数据、调用模拟器与仪器、结构化记录、评测——相当于「AI4S 版的 LangChain / Exa / MCP」。

国外：SciAgents 等科研 agent 框架(多学术);尤其值得注意 LAP(Agent-to-Instrument Protocol)(arXiv 2606.03755)——把 MCP 思想搬到实验仪器,让 agent 标准化地控制设备,目前是极早期的新标准 🟡。

中国现状：几乎空白;国内 agent 工具链(Coze 等)面向通用,不接科学工具/仪器。

中国切口 🟢：做「给科研 agent 用的工具/协议层」——① 领域语义检索(接 CNKI/万方/专利/材料库的「科研版 Exa」,墙内语料是护城河);② 实验仪器的 MCP/LAP 适配层,让 agent 能调国产仪器。这与 ④ 的 lab OS 高度协同,可一起做。

⑥「计算 + 验证」按 outcome / 里程碑收费(CRO-like) 🟡

是什么：不卖软件 license,而是用 AI 算 + 湿实验验证,按「交付了通过验证的结果/里程碑」收费的垂直 CRO-like 服务。

国外/中国头部：深势、晶泰(已上市盈利)、药明(传统 CRO + AI)都在做平台型/全栈,已较拥挤 🟡。

中国切口 🟡：头部已占住通用全栈,新进入者别全栈——挑一个单一高价值、可客观验证的细分(某类正极/电解液、某难成药靶点、某半导体材料),做「AI 设计 + 第三方/自有湿实验验证 + 按 outcome 计费」的窄而深服务,绑上游开源模型与算力、轻资产起步。这是把卖铲子(①③④⑤)整合成业务交付的商业模式选项。

给中国创业者的 3 条收敛建议

怎么选 ① 优先「结果可被客观验证」的铲子。 ③评测、④中间件、⑤工具链门槛在工程与公信力、不赌科学运气,最适合小团队冷启动;①数据次之(要专家网络);②⑥更重(算力/湿实验)。

② 绑国产模型 + 算力 + 仪器生态。 站在深势 DeePMD/DPA、玻尔算力、国产实验仪器肩上做「最后一公里」,把「中国独有」(墙内语料、本土仪器、制造成本、国产替代政策)做成护城河。

③ 用 outcome 收费、别卖软件 license。 把 ①③④⑤ 的能力,最终包成「按通过验证的结果/里程碑收费」的窄垂直服务(⑥),现金流稳、客户认账。
最值得先挖的两格(我的判断) ④ 实验室中间件 / lab OS(中国硬件强、软件空,且是自驱实验室真瓶颈)与 ⑤ 仪器 MCP/LAP 适配 + 科研版 Exa(全球都极早、和你之前看的 MCP / agent 主题一脉相承)——这两格协同性强,可一起做成「国产自驱实验室的软件层」。
风险与使用提醒 卖铲子的天花板受「金矿有多大」约束——AI4S 整体仍在「能不能真做出科学突破」的全局问号下;铲子需求依赖淘金者持续投入。本页玩家/估值/市场数字多为媒体或自述口径(🟡/🔴)、未逐一独立验证;创业决策请回一手信息再核。
