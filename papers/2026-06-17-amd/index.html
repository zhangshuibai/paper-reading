①公司是什么：使命 / 愿景 / 商业模式

Mission / Vision

AMD 是一家无晶圆厂（fabless）半导体设计公司，自述使命是「打造端到端的高性能与自适应计算引擎，推动 AI、数据中心、PC 与边缘计算」🔴。CEO Dr. Lisa Su 在 FY2026 Q1 电话会中明确：「Data Center 现在是我们营收与利润增长的首要驱动力」🟢——这句话是理解今天 AMD 的关键：它已不再是那个「便宜的 Intel 替代品」，而是把全部叙事押在 AI 数据中心算力 上的成长股。

怎么赚钱（business model）：四大板块

AMD 设计芯片、外包给 TSMC（台积电） 代工，卖给云厂商、服务器 OEM、PC 厂商和工业客户。FY2025 营收按四个 reportable segment 拆分 🟢 10-K：

  • Data Center（数据中心，$166 亿 / 48%）——增长与故事核心。两条腿：EPYC 服务器 CPU（持续从 Intel 抢服务器份额）+ Instinct MI 系列 GPU（AI 训练/推理加速器，正面硬刚 NVIDIA）。
  • Client（PC 处理器，$106 亿 / 31%）——Ryzen 桌面/笔记本 CPU。
  • Gaming（游戏，$39 亿 / 11%）——主机半定制芯片（PS5/Xbox）+ Radeon 显卡；处于主机周期尾部、同比承压。
  • Embedded（嵌入式，$35 亿 / 10%）——2022 年 ~$490 亿收购 Xilinx 带来的 FPGA / 自适应 SoC，工业/通信/汽车客户，高毛利但近年周期性疲软。

单位经济：整体 GAAP 毛利率 FY2025 约 50%、Q1'26 升至 53%；non-GAAP 52%→55%。GPU（Instinct）的毛利率结构性低于公司均值、更显著低于 NVIDIA 的 ~71%，是估值上的关键扣分项（详见 ③④）。

②行业与竞争对手

AI 算力是当下半导体最大的钱池，玩家分三类：商用 GPU（NVIDIA 绝对霸主、AMD 第二）、服务器/PC CPU（AMD vs Intel）、定制 ASIC/XPU（Broadcom、Marvell 帮云厂商自研加速器，分流通用 GPU 份额）。所有人都依赖 TSMC 先进制程——这是全行业共同的单点风险。下面挑 7 个决定性维度对照（目标公司加粗）：

读矩阵：AMD 的卡位是「唯一一家同时在服务器 CPU（抢 Intel）和 AI GPU（追 NVIDIA）两条战线都有真实放量的厂商」。CPU 侧它是赢家、对 Intel 形成持续份额压制；GPU 侧它是清晰的「第二名」，靠开放生态（ROCm vs CUDA）和大客户（OpenAI/Meta）分流 NVIDIA 的需求外溢。最大威胁来自两个方向：① NVIDIA——CUDA 生态、71% 毛利率、Blackwell/Rubin 节奏，是 AMD GPU 故事最大的天花板；② Broadcom/Marvell 的定制 ASIC——它们帮云厂商「自研」加速器，可能在通用 GPU 之外分走一块本属于 AMD/NVDA 的蛋糕。

诚实标注：AMD 最大软肋 = NVIDIA 的 CUDA 生态 NVIDIA 二十年积累的 CUDA 软件栈（库、框架、开发者习惯）是 AI 训练事实标准。AMD 的 ROCm 在快速追赶、且开放（云厂商出于「不想被单一供应商锁死」的动机愿意扶持 AMD），但「软件成熟度 + 开发者迁移成本」仍是 AMD GPU 难以拿到 NVIDIA 级毛利率与份额的根本原因。

③财务体检

定调：健康且加速的盈利成长股——不是 pre-revenue 叙事股。资产负债表稳健（净现金 ~$9B、债务极低）、自由现金流强劲。关注点不在「会不会倒」，而在「增长质量」：数据中心 GPU 能否持续放量、毛利率能否随高端 GPU 占比提升。所有数字优先取 SEC 申报，口径标注 GAAP/non-GAAP 与财季。

盈利质量 / 现金（成熟公司视角，替代 runway）

AMD 不缺钱：现金及短期投资 $123 亿、总债务仅 $32 亿、净现金 ~$90 亿，FY2025 FCF $55 亿、Q1'26 单季 FCF $25.7 亿。资产负债表健康，没有摊薄生存压力。真正的「质量」拷问在两点：① GAAP 与 non-GAAP 差距大（FY25 non-GAAP 净利 $68 亿 vs GAAP $43 亿，差额主要是 Xilinx 收购相关的无形资产摊销与股权激励），看估值时务必区分口径；② warrant 稀释——OpenAI + Meta 两笔协议各授予最多 1.6 亿股 warrant，若全部 vest 合计最多 ~3.2 亿股，相对当前 ~16.3 亿股是 ~20% 的潜在长期稀释（按里程碑与股价目标分批 vest、且换来的是巨额 GPU 订单——是「用股权换收入」的战略性稀释，但仍是真实摊薄）。

④估值

一句话：AMD 现在按「AI 成长股」定价，不是按「半导体周期股」定价。截至 2026-06-18 收盘 $537.37、市值 ~$876B，过去 52 周 +323%。

⑤多空论点（都写到最强）

Bull thesis 看多
  • 数据中心 GPU 已是可信的第二增长引擎：Q1'26 Data Center $57.8 亿（+57%），占总营收 ~56%，且同时拿到 OpenAI（6GW，2025-10）和 Meta（6GW，2026-02）两笔超大客户协议，MI450 自 2026H2 起爬坡——这是从「单一 GPU 故事」变成「多家顶级客户背书」的质变。🟢 8-K/IR
  • EPYC 持续从 Intel 抢服务器 CPU 份额：CPU 侧 AMD 是明确赢家，Intel 自身承认误判数据中心 CPU 需求、Foundry 持续巨亏，AMD 份额仍在上行。
  • 开放生态的「反 NVIDIA 锁定」需求：云厂商不想被单一供应商垄断，有动机扶持 AMD 作为第二供应商；ROCm 在快速追赶。
  • 估值相对增速不算疯：forward PEG（按 non-GAAP）~1.2，低于部分卖方给的板块中位；若 2027 AI 营收兑现，~62x forward 会被增长「消化」。🟡
  • 资产负债表干净：净现金 ~$90 亿、强 FCF，可持续投入研发/回购，无生存风险。
Bear thesis 看空
  • NVIDIA 的 CUDA 生态 + 71% 毛利率是结构性碾压：AMD GPU 毛利率结构性低于 NVDA，软件迁移成本让 AMD 难拿到 NVDA 级份额/定价；NVDA Blackwell→Rubin 的节奏可能持续吃掉增量需求。这是 AMD 故事最根本的天花板。
  • 估值已 price-in 大量未兑现的 AI 营收：~179x trailing / ~62x forward、EV/Sales ~23x——市场把「MI450/大客户全兑现」当成既成事实，任何爬坡延期/良率问题都可能触发剧烈回撤。
  • 定制 ASIC（AVGO/MRVL）分流：超大规模客户自研 XPU 可能在通用 GPU 之外切走份额，长期压制 AMD/NVDA 的 TAM。
  • 客户与代工集中：增长高度依赖少数超大客户（OpenAI/Meta）+ 全部先进制程依赖 TSMC（地缘单点）。warrant 还带来最多 ~3.2 亿股（~20%）潜在长期稀释。
  • 周期性 + 政策风险：半导体本身有周期；MI308 已因美国对华出口管制计提 ~$8 亿减值（Q2'25），中国市场政策反复是真实利润扰动。
  • 「已签 vs 预期」要分清：Meta/OpenAI 协议是多年框架 + 按里程碑 vest 的 warrant，不是锁定的硬订单收入；花旗 $330 亿/2027、$508 亿/2028 GPU 营收是卖方预测，利益相关，须打折看。🔴
花旗报告作为多头数据点（🔴 卖方、利益相关） 花旗分析师 Atif Malik 于 2026-06-12 将 AMD 由 Neutral 上调至 Buy、目标价 $460→$575 🔴。核心逻辑是把 Meta 6GW 合作（custom MI450）按「每 GW ≈ $150 亿营收」直接折算，并据此把 2027 AI GPU 营收上调至 ~$330 亿、2028 ~$508 亿，用 sum-of-the-parts 把数据中心 GPU 业务单独估到 ~$281/股。务必当作「最乐观一档假设」看：这是卖方预测、利益相关，且把「框架协议/warrant」近似当成确定营收；与 ④ 的 Bull 情景同源，不构成基本面已兑现的证据。

⑥催化剂 & timeline

  • 每季财报：Data Center 营收与 GPU 放量是否兑现指引（Q2'26 指引 ~$11.2B、non-GAAP 毛利率 ~56%）。🟢
  • MI400/MI450 量产与良率（2026H2）：故事兑现的第一道闸门——延期/良率是最直接的 thesis-breaker。🔴 指引
  • OpenAI / Meta 首个 1GW 爬坡（2026H2→2027）：把「框架/warrant」转成真实营收的验证。
  • warrant 里程碑 vesting：首批 vest 既是利好信号（订单兑现），也是摊薄落地。
  • 新品发布 / NVDA Rubin 节奏：竞争格局变化。

⑦风险因素（10-K 提炼，按重要性）

  • 竞争 / NVIDIA 主导：AI 加速器市场被 NVIDIA 凭 CUDA 生态主导，AMD 在软件成熟度、份额与定价上处于追赶位；若 ROCm 生态扩张不及预期，GPU 增长与毛利率均受限。🟢 10-K
  • 客户集中：增长越来越依赖少数超大规模 AI 客户（OpenAI、Meta、主要云厂商）；任一大客户削减资本开支或转向自研 ASIC，都会显著冲击预期。🟢
  • 对 TSMC 的代工依赖：先进制程几乎全部外包给 TSMC，产能、价格与台湾地缘风险是全行业共同的单点。🟢
  • 出口管制 / 中国市场：美国对华先进 AI 芯片出口管制已造成实际损失——MI308 在 FY2025 Q2 计提约 $8 亿存货及相关减值；政策反复直接扰动利润与可寻址市场。🟢
  • warrant 稀释：OpenAI + Meta 协议各授予最多 1.6 亿股 warrant（合计最多 ~3.2 亿股、~20%），按里程碑/股价目标 vest，长期摊薄真实存在。🟢 8-K
  • 周期性与高估值错配：半导体需求有周期，而当前估值（~62x forward / ~23x EV/Sales）已计入持续高增长，AI 资本开支若降温，估值与盈利可能同时下修（戴维斯双杀风险）。🟡
  • 整合与无形资产：Xilinx（及后续并购）带来大额摊销，GAAP 盈利被压低、GAAP/non-GAAP 差距大。🟢
