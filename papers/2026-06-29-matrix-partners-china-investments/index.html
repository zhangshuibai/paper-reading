① 背景速补：经纬是谁、张颖、与 Matrix US 的关系、风格 who

主体：经纬创投 / 经纬中国 / MPC（Matrix Partners China）

经纬创投（中文亦称「经纬中国」，英文 Matrix Partners China，简称 MPCi / MPC）成立于 2008 年，总部在北京，是中国本土早期到早成长期的头部美元 + 人民币双币基金。创始团队为 张颖（Richard Zhang，创始管理合伙人）、邵亦波、徐传陞 等。🟢 经纬官方/百科口径；🟡 维基

规模上，多方口径为管理规模超 700 亿元人民币（约 96 亿美元，2024 年），团队约 120 人，累计投资 800+ 家企业。2023 年 7 月完成第七期基金募集，规模约 16 亿美元，被称为当年规模最大的专注中国的基金。代表性历史被投包括小鹏汽车、理想汽车、滴滴、饿了么、陌陌、极米、蓝箭航天、星际荣耀等。🟡 维基/媒体；🟢 经纬官方

⚠️ 重要消歧：与美国 Matrix Partners 的关系。 经纬中国历史上与美国 Matrix Partners 同源（中文资料一度称其为美国 Matrix 的「中国子公司 / 关联团队」），但运营、决策、募资长期独立。2024 年 7 月 1 日，中国实体将品牌更名为 MPC（印度团队更名 DZ47，美国团队继续用 Matrix 名义），经纬方面表述为「体现本地化策略与各区域团队的组织独立性」，并强调「一开始就是独立的主体和团队……不存在拆分」。

实务结论：本文只统计「中国实体（经纬创投 / MPC）」参与的投资，不计美国 Matrix Partners 的美国本土项目。凡来源未能区分中美主体的，本文标 🔴 存疑 或不予收录。各方对「拆分 / 独立 / 子公司」用词不一致——这属于口径分歧，本文如实标注而非裁定。🟢 经纬官方更名说明（stcn 报道引述）；🟡 维基「子公司」表述

风格速写

  • 早期为主、连续加码：偏好天使 / Pre-A 阶段进入，对看好的公司在 Pre-A+/Pre-A++ 等多轮持续跟投（典型：钛虎机器人、西恩科技、银河通用）。🟡 媒体盘点
  • 硬科技底色重：经纬自述「100% 助力科技创新、90%+ 为硬科技项目」；2025 年初被投中 AI 类约占 37%，但相较同行更偏商业化路径清晰的智能机器人而非纯基础大模型。🟢 经纬自述；🟡 创投月报
  • 赛道矩阵打法：在机器人方向同时押「本体全栈」（宇树、智元、银河通用、星尘智能）+「产业链上游」（钛虎一体化关节、光轮 / 诺亦腾数据、飞捷科思物理 AI 引擎），形成上下游联动。🟡 经纬观点稿/媒体
  • 覆盖面广：TMT / 科技、企业服务、医疗、硬科技、新消费均有布局；2025 年首次进入美妆赛道（First Cover）。🟡 媒体
名词速补（给只懂 LLM 的读者）。 具身智能（Embodied AI）：让 AI「有身体」——把感知、决策、动作放进物理机器人里，区别于只在屏幕里输出文字的语言模型；难点是真实世界数据稀缺、物理交互不可微。 物理 AI / Physical AI：为具身智能提供「世界仿真 + 合成数据」的底层引擎，相当于给机器人造一个可以无限试错的「虚拟训练场」。 一体化关节：人形机器人的「肌肉 + 关节」总成（电机 + 减速器 + 驱动 + 编码器封装在一起），是硬件成本与性能的关键瓶颈件。 Pre-A / 天使 +/ Pre-A++：早期融资轮次的细分；「+」「++」表示同一阶段的追加轮，常见于硬科技公司「小步快跑」式融资。

④ 主题观察 themes

1）下注重心：具身智能「全栈 + 上游卖铲」

2025–2026 最密集的火力在具身智能：一边在「本体全栈」公司持续加码（银河通用自 2023 种子轮起连投至 2025-12 的超 3 亿美元新轮），一边在产业链上游密集布点——物理 AI 仿真引擎（飞捷科思 Pre-A1/Pre-A2 联合领投）、具身数据（诺亦腾追投）、一体化关节 / 伺服驱动（钛虎、西恩）。这是典型的「既投造机器人的，也投给机器人卖铲子的」上下游联动打法。🟢/🟡

2）硬科技国产替代是底色

半导体存储（领德创）、驱动芯片 / 伺服（西恩，哈工大系）、汽车线控底盘（拿森）、生物基化学品（中科国生）——共同特征是「卡脖子环节 + 工程化团队 + 商业化路径清晰」，与经纬「90%+ 硬科技」的自我定位一致。🟢 经纬自述；🟡 媒体

3）AI 偏「能落地的应用 / 数据」，少碰纯基础大模型

2025 年初被投中 AI 约占 37%，但相较追逐基础大模型的同行，经纬更偏商业化前景明确的智能机器人与数据 / 仿真基础设施。换句话说，经纬的「AI 仓位」更多落在物理世界与垂直数据，而非通用 LLM 军备竞赛。🟡 创投月报观察

4）首次进入美妆 + 持续守住消费出海

First Cover（功效型彩妆，亿元天使、经纬独家）是经纬首次布局美妆；同时 Klook（旅游体验，E+ 轮 1 亿美元）延续其「消费出海」老仓位的加码。🟢/🟡

5）「小步快跑、连续追投」的早期打法

钛虎（Pre-A → Pre-A+ → Pre-A++）、西恩（天使+ → Pre-A → Pre-A+）、银河通用（种子 → 天使 → 战略 → Pre-A+ → 新轮）都体现了同一套路：早期独家或领投进入，看对了就在后续每一轮持续加码，尽量摊低成本、锁定份额。🟡 媒体盘点
