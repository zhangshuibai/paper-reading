CPU 供应链：从 ISA 到成品芯片，谁供给谁（关系图 + 数据流）

把一颗 CPU 从「指令集 + 设计工具 + IP」到「代工 + 设备 + 材料 + 封测」再到成品出货的产业链，画成可点、可播的关系图。

来源等级：🟢 一手 官方/规范/论文 · 🟡 二手 媒体/分析 · 🔴 自述/未证实； 数字多为「约/数量级/视年份」，份额/制程会随时点变动，仅标代表公司/型号。

一句话 — 造一颗 CPU 不是一家公司的事：先拿 ISA（指令集） 定「这芯片说什么语言」、用 EDA 软件 + 买来的 IP core 把电路设计出来，再把版图交给 foundry（晶圆代工厂） 流片；代工厂靠 ASML 的 EUV 光刻机 等设备和日本主导的硅片/光刻胶等材料，用数百道工序在晶圆上做出芯片，最后送 OSAT（封测厂） 切割封装测试，成品才交给 PC/服务器 OEM 和云厂。这张图能看清「谁授权/谁制造/谁供给谁」，以及链条上几个把全行业卡死的咽喉：EUV（ASML 独家）、EDA 三巨头、光刻胶/硅片（日本）、先进封装 CoWoS（TSMC）。本文仅标产业链位置，非投资建议、非背书。

①领域背景速补（半导体制造）

这篇假设你只懂 LLM、不懂芯片制造。先把链条里反复出现的专名用大白话讲清——它是什么、为什么关键、在链条哪一环。术语保留英文，旁边给中文。

ISA Instruction Set Architecture · 指令集架构
芯片能听懂的「机器语言词汇表」（加法、跳转、读内存怎么编码）。关键在于：它定义生态——同一 ISA 的软件能跨芯片跑。代表：x86（Intel/AMD 双寡头、封闭授权）、Arm（授权 IP 给别人用）、RISC-V（开放、免授权费）。类比：芯片说的「方言」，软件得用同一方言写。
IP core 知识产权核 · 可复用电路模块
别人设计好、可买来即用的电路积木（一个 CPU 核、一个 USB 控制器、一段内存接口）。关键在于：没人从零造每个模块，买 IP 拼起来更快。代表：Arm（卖 CPU 核 IP）、Synopsys、Cadence。类比：盖楼买现成的预制构件。
EDA Electronic Design Automation · 电子设计自动化
设计芯片用的「专业 CAD 软件」：画电路、仿真验证、自动布局布线、检查能不能造。关键在于：现代芯片几十亿到上千亿晶体管，没 EDA 根本画不出来，且只有三家做得全。代表：Synopsys、Cadence、Siemens EDA；国产华大九天。类比：芯片设计师的 Photoshop+CAD+编译器全家桶。
fabless vs IDM 无厂模式 vs 垂直整合
fabless（无厂）=只设计、把制造外包给代工厂（AMD、Apple、高通）；IDM（垂直整合）=自己既设计又有自己的厂（经典 Intel、过去的三星）。关键在于：建一座先进厂要几百亿美元，多数公司选 fabless。
foundry 晶圆代工厂
专门「替别人造芯片」的工厂，自己不设计产品。关键在于：先进制程几乎被 TSMC 一家垄断（约 70% 全球代工份额、Q2–Q3 2025 🟡）。代表：TSMC、Samsung、Intel Foundry、GlobalFoundries、中芯国际 SMIC。类比：芯片界的「来图代工」精密铸造厂。
tape-out / 流片 流片 · 设计定稿交付制造
设计全部完成、把最终版图「定稿交厂」开始造的那一刻。名字来自早年把数据存磁带送厂。关键在于：流片一次先进制程要数千万到上亿美元，错了就是真金白银报废，所以前面要反复验证。
wafer / 晶圆 晶圆 · 圆形硅片基底
一片高纯度单晶硅圆盘（常见直径 300mm），所有电路都在它表面一层层「印」出来，一片能切出几百到上千颗芯片（die）。类比：一张能同时印很多张「邮票」的母版。
lithography / EUV 光刻 · 极紫外光刻
用光把电路图案「印」到晶圆上的核心工序，决定线条能多细。EUV（极紫外，13.5nm 波长）是先进制程的关键，全球只有 ASML 能造 EUV 光刻机（约 100% EUV 份额 🟡，单台 DUV/EUV 价格从数千万到上亿美元，High-NA 约 3.5–4 亿美元、2025 Q4 首台交付 Intel 🟡）。类比：用极细的光「笔」在硅上写电路，笔越细线越密。
node / 制程 工艺节点
「3nm / 5nm / 7nm」这类标签，粗略代表晶体管密度与代际（如今更像营销名而非真实尺寸）。关键在于：越先进=越密=越省电越快，但越贵越难，且越靠 EUV。先进 CPU、AI 芯片都抢最先进 node。
OSAT / 封测 Outsourced Assembly & Test · 委外封装测试
把晶圆切成单颗 die、装进有引脚/焊球的封装、测好坏的厂。关键在于：AI 时代「先进封装」成了新咽喉——把多颗 die 和内存拼在一起。代表：ASE 日月光、Amkor、长电 JCET；先进封装 CoWoS 由 TSMC 主导。类比：把芯片裸片「装盒打包贴标签」并质检出厂。
先进封装 / CoWoS Advanced Packaging
不再只追求把单颗 die 做小，而是用硅「桥」把逻辑芯片和 HBM 内存并排拼在一块基板上高速通信。关键在于：AI 芯片（GPU/加速器）严重依赖它，2025–26 产能被 NVIDIA 等订满、成 AI 供应瓶颈🟡。代表：TSMC CoWoS。
本篇聚焦 / 边界讲「一颗 CPU 怎么被设计与制造出来、链条上谁供给谁」。未含：整机/AI 集群组装、个股财务、HBM/DRAM 内存供应链细节（那些在延伸阅读里）。图为示意、非穷尽；公司仅代表该环节典型玩家，不是全部。

④给小白的心智模型：把整条链想成「定制出一本书」

把造一颗 CPU 想成出版一本极其精密的「书」：

  • ISA = 这本书用哪种语言写（中文/英文/世界语）——选了 x86 就只能让懂 x86 的「读者（软件）」读；选开放的 RISC-V 像选了无版权费的世界语。
  • EDA = 排版与校对软件（Word + 拼写检查 + 印前检查）——没有它，几十亿字的「书」根本排不出来，且全行业就三套好用的软件。
  • IP core = 买来的现成章节/段落模板——Arm 卖你写好的「核心章节」，你不用每段从零写。
  • foundry（代工厂）= 印刷厂——你把排好的版（版图）送过去，它替你印；而全世界最精密的那台印刷机基本只有 TSMC 这一家开得最好。
  • 设备/材料 = 印刷厂里的印刷机和油墨纸张——最贵最关键的那台「印刷机」是 ASML 的 EUV，全世界只有它一家造；高级「油墨/相纸」（光刻胶、硅片）主要日本供。
  • tape-out（流片）= 「付印」按钮——一按下去就开印，错字是按整批废品算钱的，所以前面拼命校对。
  • OSAT（封测）= 装订 + 质检 + 贴条码出库——把印好的「书页（晶圆上的 die）」裁切、装订成册（封装）、检查合格再发给书店（OEM/云厂）。先进封装（CoWoS）像把好几本书精装合订成一套豪华版。

一句话：设计公司写稿、EDA 排版、Arm 卖章节、TSMC 印刷、ASML 提供印刷机、日本供油墨纸张、日月光/长电装订出库——缺任何一环，书出不来。

⑤依赖与相互制约（深入）：卡点为什么是卡点

这条链最有意思的地方是「谁的能力卡住谁」。下面把几条因果链串清楚（数字视年份、🟡 为主，已尽量诚实标来源）。

卡点一：EUV → 先进制程 → 谁能做最先进 CPU/AI 芯片

EUV 光刻机全球只有 ASML 一家造（约 100% EUV 份额 🟡）。没有 EUV，要做先进 node 就得用老一代 DUV「多重曝光」硬凑——更多道工序、更低良率、更高成本。于是因果链是：ASML 是否供货 → 你能否拿到 EUV → 你能否量产最先进 node → 你能否造最强 CPU/AI 芯片。TSMC 拿得到 EUV，所以坐稳先进制程；SMIC 拿不到，用 DUV 多重曝光做到 7nm 级、并传出 5nm 级（N+3）量产，但成本明显更高、良率受限、难以持续向 3nm/2nm 下探（🟡 综合 TechInsights/媒体）。

卡点二：出口管制的「三道闸」——美(EDA) + 日(材料) + 荷(EUV)

先进 CPU 的链条恰好被三国的咽喉环节卡住，构成出口管制可下手的三道闸：

相互制约因果链 ① EDA 三巨头（美）：Synopsys / Cadence / Siemens EDA 几乎垄断先进设计全流程 → 断供高端 EDA，先进芯片连「设计」都难完成。
② 高端材料（日）：光刻胶（尤其 EUV 胶）、高纯硅片由日本主导 → 即便有机器，没「油墨纸张」也印不出。
③ EUV（荷·ASML 独家）：先进节点离不开 EUV → 限制 EUV（及部分先进 DUV）出口，直接卡住最先进 node。
三闸叠加 → 设计端（EDA）+ 制造端（EUV）+ 材料端（光刻胶/硅片）同时受限 → 最先进 CPU/AI 芯片产能被牢牢卡在少数地区。

卡点三：ISA 生态门槛——封闭 vs 授权 vs 开放

选哪种 ISA，决定了你的生态与门槛：x86 由 Intel/AMD 双寡头封闭把持，外人极难拿到授权（生态最成熟、但进不去）；Arm 走授权制，交钱拿 IP，门槛中等、生态广（手机/Apple Silicon/服务器都在用）；RISC-V 开放免授权费，门槛最低、最适合绕开授权与管制，但软件生态仍在成长。因果链：选 ISA → 决定能否拿到现成 CPU 核 IP（Arm）/ 要不要自建生态（RISC-V）→ 决定产品上市速度与软件丰富度。中国多家（龙芯 LoongArch、申威自研、平头哥押 RISC-V）正是想从 ISA 这一端绕开卡脖子。

卡点四：先进封装 CoWoS → AI 芯片产能

就算逻辑芯片造出来，AI 芯片还要靠先进封装把 GPU/加速器和 HBM 内存拼在一起，CoWoS 由 TSMC 主导。2025–2026 CoWoS 产能被 NVIDIA 等大量预订、基本「卖到 2026 年都满」（🟡 综合 CEO 表态/媒体；产能从 2024 年底约 3.5 万片/月扩到 2025 年底约 7.5 万片/月仍紧）。因果链：CoWoS 产能 → AI 芯片出货量 → 整个 AI 算力供给的节奏。这也是为什么先进封装被称作「AI 时代的新咽喉」。

卡点五：中国国产替代卡在哪

综合看，中国在设计、成熟制程、传统封测上已较强，但替代主要卡在三处：EUV（拿不到 ASML）、高端 EDA（先进节点全流程）、高端设备/材料（高端量测、EUV 光刻胶等）。结果是 SMIC 大致困在 7nm 级附近（向 5nm 级推进但成本/良率受限），先进 node 与海外差代际（🟡）。
