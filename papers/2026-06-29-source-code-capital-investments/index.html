①背景速补：源码资本是谁

主体与创始人

源码资本（Source Code Capital）由曹毅（Cao Yi）于 2014 年春创立，总部北京。曹毅此前在红杉资本中国（今 HongShan / 红杉中国）任职，是红杉体系出身的投资人，独立创业后建立源码。机构定位横跨早期到成长期，主投科技驱动的商业变革，最为人熟知的早期战绩是字节跳动与美团——这两笔奠定了源码在中国 VC 圈的地位。🟢/🟡

规模与版图

  • AUM 约 70 亿美元（约合 500 亿元人民币，2025 年 11 月口径）。🟡
  • 自 2014 年创立以来累计投资超 300 家创业公司。🟡
  • 行业覆盖：产业数字化、人工智能、机器人、先进制造、绿色发展（新能源）、企业软件、生命科学、新消费与新品牌、出海（Global）。🟢/🟡
  • 代表案例（历史 + 近期，机构官方/媒体口径）：字节跳动、美团、贝壳找房、理想汽车、悦刻（雾芯科技 / RLX）、布鲁可、塔斯汀、元保数科、OPay、Stori；以及 AI / 机器人新星宇树科技、银河通用、加速进化、月之暗面、LiblibAI、Sand.ai、奕斯伟材料、壁仞科技、云鲸智能、卧安机器人等。🟡
领域背景速补（读者只需 LLM 基础）
  • VC / 成长期投资：风险投资基金把出资人（LP）的钱投进未上市公司换股权，靠公司被并购或 IPO 退出赚钱。「早期」指天使/A 轮（公司很小、风险高、单笔金额小、占股多）；「成长期」指 B 轮之后（公司已验证、估值高、单笔金额大）。源码这次把成长期与早期拆成两条线运营。
  • 双币基金：同时募集人民币和美元两种货币的基金。美元基金能投/退海外与红筹结构，人民币基金对接 A 股科创板和国资 LP。中美环境收紧后，能两边都跑通是头部机构的护城河。
  • 领投 vs 跟投 / 老股东加注：「领投」是本轮定价、出资最多、谈条款的主导方；「跟投/参投」是搭车；「老股东持续加注（增持）」指此前已是股东、新一轮继续按比例或超额追加——这是源码在很多明星项目里的真实角色，不等于本轮领投，本文严格区分。
  • 具身智能 / 人形机器人：让 AI 不止「在屏幕里说话」，而是装进有身体的机器人去抓取、行走、干活；是 2024–2026 中国一级市场最热的赛道之一，源码重仓宇树、银河通用、加速进化等。
  • 科创板 / 港股 IPO：上交所科创板（STAR Market）面向硬科技、允许未盈利企业上市（「科创板八条」「新国九条」放宽了规则）；港股是中国科技公司另一上市地。两者都是 VC 的主要退出通道。
  • AI2C / 2P：源码新基金的提法。2C = 面向消费者的 AI 应用；2P（to Prosumer / 专业个人）= 面向有付费能力的专业/创作者个体（如 AI 创作工具）。对应它对 LiblibAI、Sand.ai 这类「AI 应用 / 创作工具」的偏好。

2025–2026 的两件大事

(1) 新一期成长基金落地（2025 年 11 月）。源码完成新一期双币成长基金募资，总规模约 6 亿美元，投资期 5 年、退出期 20 年、整体存续周期约 25 年；资金「主要来自产业投资人和少数长期财务投资人」。规模相比 2021 年那一期（10 亿美元 + 70 亿元人民币）有所收缩，被解读为转向「少而精」——单项目下注更大、周期更长、加重投后。主题明确为「AI+」（AI2C/2P、算力产业链、卡脖子突破、AI 规模商业化）与「Global+」（智能硬件、软件应用出海）。🟢/🟡

(2) 早期与成长期分工。由黄云刚独立运营的「源码律动」明确 all-in 早期、回归经典 VC 打法；新成长基金侧重成长期。换句话说，2025 年起源码把「找早期项目」与「成长期重仓」拆成两套班子。🟡

④主题观察

1) 从「互联网+」彻底切到「AI + 硬科技」

源码的历史标签是字节、美团、贝壳、理想这类「互联网/出行/消费平台」。2025–2026 公开曝光的重心已明显转向大模型、具身机器人、国产算力/半导体。新基金把主题直接写成「AI+」与「算力产业链/卡脖子」，与组合里的壁仞（GPU）、奕斯伟（大硅片）、银河通用/宇树/加速进化（机器人）形成呼应。🟡

2) 「老股东持续加注」是源码近期最典型的动作

在加速进化、云鲸、卧安、月之暗面、LiblibAI 等多个项目里，源码的角色都是早期进入 + 后续多轮加码，而非每轮领投。这与新基金「少而精、单项目下注更大、加重投后、长期陪伴」的自述一致——把弹药集中压在已验证的老组合上，而不是广撒网。🟡

3) AI2C/2P：偏好「AI 应用 / 创作工具」而非只押底层模型

LiblibAI（AI 图像创作社区）和 Sand.ai（视频生成 + VidMuse 产品）都属于「模型 + 产品双轮」的应用层，呼应新基金对 AI2C/2P（消费者 / 专业创作者）的明确点名。即便同时持有月之暗面这类底层模型，源码在应用层的下注同样显眼。🟡

4) 募资收缩 + 早晚期分线，反映「降速提质」

新一期成长基金 6 亿美元，低于 2021 年那期；同时把早期单拆给「源码律动」。组合拳的信号是：成长期更聚焦、单笔更重、周期更长，早期则回归经典 VC 小额多投的打法。这是行业整体募资降温背景下头部机构的常见调整。🟡

⑤退出 / IPO（2025–2026 样本）

2025–2026 是源码硬科技组合的集中收获期，三笔可核实的 IPO 退出窗口：

  • 奕斯伟材料（西安奕材，688783）：2025-10 登陆上交所科创板，募资约 46.36 亿元，系「新国九条/科创板八条」后首家受理并上市的未盈利企业，首日大涨。源码为上市前机构股东之一。🟢/🟡
  • 卧安机器人（06600.HK）：2025-12-30 在港交所挂牌，发行价 73.8 港元、首发募资约 16.4 亿港元，号称「AI 具身家庭机器人第一股」。源码上市前为最大机构股东之一。🟡
  • 壁仞科技（Biren，06082.HK）：2025-12 递表、2026-01 上市，募资约 6.23 亿美元，系「港股 GPU 第一股」。源码为 2021 年 B+ 轮参投方之一。🟡
提示 上述为「公司 IPO 事件」，源码作为股东可借此实现部分退出，但源码具体持股比例、解禁安排与实际兑现收益均未公开披露，不应据此推断回报。

⑥局限与存疑

  • 样本性。本文是公开披露样本，非完整组合。源码累计投资超 300 家，本文仅覆盖能被新闻/公告核实的十余笔，存在严重的「曝光偏差」（有 IPO/大额融资的项目更易被收录）。
  • 角色易混淆。「源码为股东/老股东」≠「源码本轮领投」。很多明星项目近期大额轮由国资/产业方领投，源码只是早期股东或跟投；文中已尽量区分，但部分项目份额/轮次确实未披露。
  • 金额与口径。一级市场金额多为媒体口径，可能含估值传闻、未必经官方确认；人民币/美元换算、AUM「约 70 亿美元 / 约 500 亿元」等存在口径与时点差异。
  • 时间边界模糊。云鲸等项目的融资披露时点跨 2024 末–2025，月之暗面/银河通用为连续多轮，难以精确切到单一日期。
  • 未覆盖。企业服务/SaaS、新能源、生命科学、消费新品牌等赛道，源码历史上均有布局，但本样本期内未核实到可靠的「新投/重大事件」，故清单中较弱——这是核实缺口，不代表源码没在投。
  • 消歧。「源码资本」≠ 红杉中国（HongShan）≠ 顺为 ≠ 明势；曹毅虽红杉出身，但源码为独立机构。请勿与同期红杉中国盘点混淆。
