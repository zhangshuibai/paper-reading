一句话 — Unreasonable Labs 是 2026-03 刚出 stealth 的「AI for 科学发现」初创，口号「Build superintelligence for autonomous knowledge creation and discovery」——做一个能跨学科组合出新知识、生成新假设（而非检索已有事实）的 AI 发现引擎。产品是 Unreasonable.DISCOVERY。创始团队是明星配置：Yuan Cao（CEO，前 DeepMind、Gemini 核心贡献者）+ Markus Buehler（MIT 教授，AI 材料科学、SciAgents 作者）。技术血统 = 知识图谱 + 多 agent 图推理（SciAgents）+ 范畴论式生成 + 前沿 LLM。属「明星团队 + 强叙事 + 产品实效待验证」阶段。

①是什么

一家做「autonomous knowledge creation and discovery（自主知识创造与发现）」的 AI 公司。官网两句核心：

“Build superintelligence for autonomous knowledge creation and discovery.”
“AI that discovers new knowledge and solves the hardest problems across domains, making experts faster and smarter.”

关键词是「新知识 / 跨学科 / 生成假设」——强调不是 RAG 式检索已有事实，而是组合出此前没人连起来的连接。目标用户：科学家（基础研究）、工程师（材料 / 生物 / 能源 / 半导体）、企业战略与创新分析师、研究员（假设生成、文献综合）。

③产品：Unreasonable.DISCOVERY

官网定位「An AI discovery engine that helps you compose new knowledge across disciplines」。由三块组成：

  • Unified World Model：把跨领域数据整合成互联的「世界表征」——本质是大规模知识图谱式的统一表示。
  • Insight Generation & Deep Reasoning：多步推理生成新假设，官网明确「rather than retrieve existing facts」。
  • Human-AI Co-Discovery：透明、可被科学家引导与批判的协作（呼应「可解释、非纯黑箱」）。

注截至调研日，产品为官网描述层级；未见公开 demo、定价、客户案例或第三方 benchmark。

④有什么 research（技术血统）

公司刚出 stealth，暂无公司署名论文。但 Unreasonable.DISCOVERY 的技术原型，就是联创 Markus Buehler 在 MIT 的研究线——把它对回官网的「LLM + 关系图谱 + neurosymbolic 抽象」，几乎是逐条对应：

研究 / 系统核心思想（= 产品原型）出处
SciAgents多 agent + 知识图谱推理自动做科学发现：大规模 ontological 知识图谱 + 一组 LLM/检索工具 + 带 in-situ 学习的多 agent，「模拟一个科学家社区做发现」，在 bio-inspired 材料上揭示跨学科隐藏关系arXiv 2409.05556（Advanced Materials 2025）· 开源
Graph-based generative discovery范畴论（category theory）启发的图生成 AI：靠「揭示隐藏连接」拿到更高 novelty / 探索性 / 技术细节arXiv 2403.11996（Mach. Learn. Sci. Technol.）
Yuan Cao 一侧前沿 LLM 工程（Gemini、大规模训练 / 推理）——补上「强推理大模型」这条腿Google DeepMind（Gemini）
读法 把官网三组件翻译成学术语言：Unified World Model ≈ SciAgents 的 ontological 知识图谱；Insight Generation ≈ 多 agent 图推理 + 范畴论式生成（找隐藏连接）；Human-AI Co-Discovery ≈ 多 agent 里把人放进 critique 环。再叠 Yuan Cao 的前沿 LLM。所以它的 research 不是空的——是把一条已发表、且开源过原型（SciAgents）的学术路线，做成产品。

⑤现实判断

  • 定位独特性：在「AI 做科学假设」这条赛道里，它走的是知识图谱 / 范畴论 + 多 agent路线——区别于 Google AI co-scientist 的「生成-辩论-锦标赛」纯 LLM 路线，也区别于 A-Lab 的 wet-lab 机器人闭环。卖点是「跨学科组合出隐藏连接」+ neurosymbolic（想要可解释）。
  • 团队是最大资产：Gemini 核心 + MIT 材料 AI 权威 + 诺奖/Langer/HF 顾问，叙事与背书都顶配。
  • 诚实的保留：非常早期（种子轮、刚露面）。目前能核实的是「团队 + 方向 + 学术原型」；产品实效、是否真做出过新发现，暂无第三方证据。「discover new knowledge / superintelligence」是强叙事，需用时间和案例检验——这也正是整条「自主科学」赛道的通病（参见本库的科学发现综述与 A-Lab 争议）。
