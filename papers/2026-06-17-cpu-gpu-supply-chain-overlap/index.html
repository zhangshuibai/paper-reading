CPU 与 GPU 供应链的重合关系：哪些共享 / 哪些专属 / 为什么互相挤产能

CPU 和 GPU 是两种芯片，却抢同一批上游卡点的产能——一张关系图看清「共享层 / CPU 专属 / GPU 专属」，再用动画演示 AI 浪潮如何把共享产能从 CPU 那边吸走。

来源等级：🟢 一手 官方/规范/财报 · 🟡 二手 媒体/分析 · 🔴 自述/未证实； 份额/产能/制程数字多为「约 / 数量级 / 视年份」，并尽量标代表公司。

一句话 — CPU 和 GPU 看着是两类芯片，但它们的上游供应链大量重合：都用同一批 EDA（Synopsys/Cadence/Siemens）、挤同一个 TSMC 先进制程、共用同一批 ASML EUV 设备与硅片/光刻胶/特气材料、走同一套 CoWoS/chiplet 先进封装与 OSAT 封测。正因为共享这些卡点（bottleneck），AI 把 GPU 需求拉爆时，就会直接从 CPU 那边吸走产能、抬高整链成本、并强化这几个卡点的定价权。本篇用一张三栏关系图 + 一段「产能争夺」动画把这件事讲清。

①领域背景速补（给只懂 LLM、不懂半导体的读者）

这篇讲的是芯片产业链，里面全是缩写。先把关键名词讲清「它是什么 / 为什么难 / 和这篇有什么关系」，后面看图就不卡了。术语保留英文，旁边给中文。

  • CPU（中央处理器）：电脑/服务器里管「通用逻辑、调度、低延迟串行任务」的芯片，核心少但每核很强、擅长分支预测。代表设计公司 Intel、AMD、Apple、Qualcomm、以及 Arm 阵营。它和这篇的关系：CPU 是「需求方一」。
  • GPU（图形/通用并行处理器）：原本画图、现在是 AI 训练/推理的主力。特点是「海量并行」——成千上万个小核同时算矩阵乘。代表 NVIDIA、AMD。它和这篇的关系：GPU 是「需求方二」，而且 AI 让它的需求暴涨。
  • 为什么 CPU 和 GPU 会「抢」？ 因为它们虽然算的东西不同，但造出来的方式高度雷同——都要先用软件设计、再到同一家代工厂用同一批设备和材料造出来、再封装测试。产业链上游是「共享的」，所以谁需求大谁就把产能占走。
  • EDA（Electronic Design Automation，电子设计自动化软件）：设计芯片的「IDE + 编译器 + 仿真器」。没有它没人能把上百亿晶体管画出来并验证。三家垄断：Synopsys、Cadence、Siemens EDA。CPU 和 GPU 都离不开它，是第一个共享卡点。
  • Foundry / 晶圆代工（Foundry）：只负责「制造」别人设计好的芯片的工厂。TSMC（台积电）是全球先进制程的绝对龙头（约 70% 代工份额，先进节点更高 🟡）。关键点：AMD 的 CPU、NVIDIA 的 GPU、Apple 的芯片都在 TSMC 同一批产线上造——这是最大的重合点。
  • 制程节点（process node，如 5nm / 3nm / 2nm）：越小代表晶体管越密、越省电越快。「先进制程」指 7nm 及以下。越先进的产线越稀缺、越贵，CPU 和 GPU 都想用最新的，于是抢。
  • EUV（Extreme Ultraviolet，极紫外光刻）：造先进芯片必须的「曝光机」，把电路图投到硅片上。全世界只有 ASML（荷兰）一家能造 EUV 光刻机——这是设备端的单点卡脖子。所有先进 CPU/GPU 都依赖它，外加 AMAT / Lam / KLA 等设备厂。
  • 材料：硅片（wafer）/ 光刻胶（photoresist）/ 特气（specialty gas）：造芯片的「原料」。硅片是芯片的「画布」（日本信越/SUMCO 等主导），光刻胶是感光涂层（日本 JSR/TOK 等），特气是工艺用气体。CPU/GPU 共用同一批料。
  • 先进封装（advanced packaging）：CoWoS / chiplet / Foveros：芯片造好后，把多块小芯片（die）拼到一起、或把芯片和内存堆在一块基板上的「高级组装」工艺。
    • CoWoS（TSMC 的招牌封装）：把 GPU 主芯片和 HBM 内存并排放到一块硅中介层（interposer）上。AI GPU 重度依赖它，也是当前最紧的卡点之一。
    • chiplet（小芯片）：把一颗大芯片拆成几颗小芯片再拼起来，良率更高、更灵活。高端 CPU 也大量用（AMD 的 EPYC/Ryzen、Intel）。
    • Foveros：Intel 的 3D 堆叠封装，把芯片上下叠。
  • HBM（High Bandwidth Memory，高带宽内存）：把多层 DRAM 垂直堆叠、紧贴 GPU 的超高带宽内存。GPU/AI 重度依赖 HBM（喂数据给上万个核），而 CPU 大多用便宜得多的 DDR 内存——这是 CPU/GPU 一个重要的「专属」分界。HBM 三家主导：SK海力士（SK Hynix）、Samsung、Micron（美光）。
  • OSAT（Outsourced Semiconductor Assembly and Test，封测外包）：专门做「封装 + 测试」的工厂（如日月光 ASE、Amkor）。CPU/GPU 共用。
  • IP / Arm：可复用的电路设计「积木」（如 Arm 的 CPU 核、各种接口 IP）。CPU 和不少 SoC/GPU 都可能买来用。
  • 卡点 / 瓶颈（bottleneck）：产业链上「只有一两家能做、且短期扩不出产能」的环节。本篇核心就是：CPU 和 GPU 共享了好几个卡点（TSMC 先进制程、ASML EUV、CoWoS、HBM），所以会互相挤。

说明：这是「示意、非穷尽」的产业链骨架；真实供应链还有更多层级（基板、载板、测试设备、化学品……），本篇聚焦「CPU 与 GPU 的重合与争夺」这条主线。

④给小白的心智模型

一家米其林后厨，同时接 CPU 桌和 GPU 桌两桌订单。

把整条供应链想成一家顶级餐厅：

  • 菜谱设计软件 = EDA：所有菜都得用这套软件设计配方，全世界只有三家供这软件。
  • 后厨（同一个） = TSMC 代工厂：CPU 桌点的菜、GPU 桌点的菜，都在这同一个后厨、用同一批炉灶（先进制程产线）做。后厨就这么大。
  • 那台全世界独一份的高级烤箱 = ASML EUV：做任何招牌菜都得排队用它，别家造不出第二台。
  • 稀有食材（黑松露） = HBM + CoWoS 封装产能：GPU 桌的招牌菜（AI 加速卡）每一道都要大量黑松露；CPU 桌大多点用普通食材（DDR）的菜。

平时两桌相安无事。但 AI 这桌（GPU）突然一口气点了 100 份招牌菜——它把后厨的炉灶、那台独一份烤箱、和几乎所有黑松露全占了。结果：CPU 桌的菜上得更慢（交期拉长）、价格被抬高（产能稀缺涨价）、而供炉灶/烤箱/松露的几家供应商说一不二（定价权变强）。这就是「因为共享后厨，所以 GPU 的暴涨直接挤了 CPU」。

⑤依赖与相互制约（深入）：为什么「共享」就一定「互挤」

把链条接口讲清：谁靠什么连谁、谁的能力卡住谁。核心逻辑是——CPU 和 GPU 在同一批有限产能的卡点上是「零和」关系：这些环节短期扩不出产能（建一座先进 fab/封装厂要数年、ASML EUV 年产有限），于是一方多拿，另一方就少拿或更贵。

三个共享瓶颈，被 AI/GPU 拉满

  • TSMC 先进制程：AMD CPU、NVIDIA/AMD GPU、Apple SoC 都排在 TSMC 同一代先进节点（如 3nm/2nm）上。晶圆产能（wafer/月）短期固定，AI GPU 大量吃掉 wafer 配额，留给 CPU 的先进产能就被压缩，且新节点首发产能往往优先给出价最高、量最大的 AI 客户。TSMC 约 70% 代工份额、先进节点占比更高 🟡，使其定价权很强。
  • CoWoS 先进封装：AI GPU 几乎每片都要 CoWoS（把 GPU die 和 HBM 拼到中介层上）。媒体/分析称 CoWoS 产能到 2026 年前基本售罄、NVIDIA 一家占约 60%、前三大客户合计 >85% 🟡，留给其他玩家（含想用高端封装的 CPU/加速器）的份额很小，交期可长达一年以上 🟡。
  • HBM 供给：GPU 重度依赖、CPU 大多用 DDR，所以 HBM 名义上「偏 GPU 专属」，但它仍是共享内存产线与产能的一部分——HBM 用的是 DRAM 晶圆和先进封装/TSV 工艺，HBM 拉满会占用 DRAM 厂的产能与资本开支，间接抬高包括 CPU 用的 DDR 在内的整体内存价格。HBM 三家主导（SK海力士 ≈62% / Micron ≈21% / Samsung ≈17%，Q2 2025）、到 2026 前基本售罄 🟡，定价权强。
  • ASML EUV：再上一层，TSMC/Intel/Samsung 想扩先进产能都得先拿到 EUV 机台，而 EUV 全球只有 ASML 一家供 🟢，年产有限。这把「扩产」本身卡死在最上游：不管 CPU 还是 GPU 想要更多先进产能，最终都撞到同一台「独一份烤箱」。
相互制约因果链 AI 浪潮 → 云厂/大模型疯抢 GPU（NVIDIA/AMD） → GPU 订单灌向 TSMC 先进制程 + CoWoS 封装 + HBM（三个共享卡点）→ 这些卡点产能短期固定、扩产要数年且受 ASML EUV 年产上限约束 → 于是 GPU「拿满」后，留给 CPU 的先进晶圆/封装份额被挤压、整链报价上调、交期拉长 → 同时几个卡点（TSMC/CoWoS/HBM/ASML）因为「谁都绕不开」而定价权与利润率走强。一句话：因为共享，所以 GPU 的需求暴涨会沿着供应链「传导」成 CPU 的成本与交期压力。

CPU 与 GPU 的边界正在模糊

「CPU 专属 / GPU 专属」并不是铁墙，反而越来越交融，这让「重合」更深：

  • GPU 里有 CPU：现代 GPU/加速器内部也有控制逻辑；数据中心一块 GPU 旁常配 CPU 做调度。
  • CPU 里加 AI 单元：新一代 CPU（含 NPU/矩阵扩展）也开始做 AI 推理，吃更先进的制程。
  • APU / 超级芯片：AMD 的 APU、NVIDIA 的 Grace-Hopper / Grace-Blackwell「超级芯片」把 CPU 和 GPU 封进同一个封装（用 chiplet/CoWoS 互联）🟡——这时 CPU 和 GPU 不只是「共享上游」，而是共享同一块基板和封装产能，争夺关系被进一步内化。
诚实边界 具体份额/产能/制程数字（如 CoWoS 60%、HBM 62%、TSMC 70%）多为 2025–2026 年的媒体与分析口径 🟡，会随年份、统计口径（按营收 vs 按晶圆 vs 按「Foundry 2.0」）、新增产能投放而变化；本图是示意、非穷尽的产业链骨架。请以 TSMC/ASML/SK海力士/Micron 等官方财报与原始报告为准。
