①按「该不该切」分三组

🟢 最像 whitespace（相对空白，值得切） ① Orbital 那条——AI 设计数据中心冷却液：中国有液冷系统厂（英维克）、有氟化液厂（巨化 / 新宙邦承接 3M 退出 PFAS 的国产替代），但没人把「AI 设计新冷却液材料 + 制造」合起来；东数西算 PUE 红线 + 无 PFAS 是双 why-now，可「AI 团队 × 在位产能合资代工」避开重资本。
② Exa 那条——给国产 agent 用的中文 search API：C 端 AI 搜索很卷，但「按调用收费、给大模型 / agent 用的 search API」国内几乎只有博查一家；护城河押在海外拿不到的微信 / 小红书 / B站墙内语料 + 合规。
🟡 有切口但要差异化 / 换打法 材料发现（CuspAI）、自驱材料实验室（Periodic）、电池界面（QuantumBond）、显示材料 AI 工具（Ares）、垂直卡脖子材料（Altrove 换品类）——共同打法是「别造通用模型 / 别赌单一重资产 IP，去做工具 / as-a-service / 贴龙头 / 换成中国真被卡的品类」。
🔴 红海 / 壁垒高，谨慎或只做卖铲子 AI 药物发现（英矽已 IPO、临床比 Isomorphic 还靠前，晶泰 / 百图 / 剂泰扎堆）、CRO 2.0（晶泰已上市盈利、份额 >30%）、极端材料 / 国防（能力在体制内、军工四证等结构性壁垒）。正面硬刚没空间，能做的多是卖铲子（数据 / 评测 / 工程化工具）或深度绑定在位者。

②三条贯穿性结论（给创业）

  1. 「卖铲子 / as-a-service」是反复出现的安全切口。 多数拥挤赛道（药物、CRO、材料平台）的中国版结论都指向：别赌单一 IP / 重资产，去做工具、数据、评测、按量服务。
  2. 中国在「预测 / 模型层」并不落后，甚至反超。 深势、晶泰、AISI、机器化学家、英矽——纯做模型 / 平台的窗口在中国也基本关了；机会在下游闭环（造 + 认证 + 合同）与垂直卡脖子品类 + 国产替代 / 地缘 why-now。
  3. 「中国独有」往往就是护城河。 墙内语料（中文 search API）、在位制造产能（冷却液代工）、CRO + 工程师红利（低成本 AI↔实验闭环）、国产替代政策——这些是海外玩家拿不到的本土壁垒。
