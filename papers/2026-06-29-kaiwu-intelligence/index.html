一句话 — 开物纪是前微软研究院 AI4Materials 负责人陆子恒于 2025 年 9 月创办的北京 deep-tech 公司，用「材料大模型 + 自动化高通量实验」做闭环，目标是把材料从虚拟「发现」推到吨级「量产」。它的关键选择是不卖铲子、自己挖金——以「自有材料 IP 管线」为主、产业合作为辅（双轨），公开对标孵化出 Moderna 的 Flagship Pioneering（venture-creation / 自己造资产）🟢。赛道判 emerging（新兴）：海外 Periodic Labs / Lila / Radical AI 同期密集融资、巨头（MatterGen/GNoME）已开源压低「预测」价值，但从实验室到量产的工程化是全行业生死线，产品实效全部未验证。已知融资：2025-11 亿元级天使轮（IDG、蓝驰领投）+ 2026-03 数亿元天使+轮（Monolith 领投）🟡。

⓪领域背景速补（只懂 LLM 也能跟上）

这家公司在「材料科学 + AI」交叉处，下面把会反复出现的术语先讲清，假设你只懂 LLM、不懂材料。

  • 材料发现（materials discovery）是什么、为什么难：材料指的是电池正极、半导体、磁体、超导体、散热/导热材料这类「有特定物理性能的物质」。难点在于组合空间天文数字（元素 × 配比 × 晶体结构 × 工艺），且「想出一个结构」和「真造出来、还稳定、还能批量做」之间隔着千山万水。过去几十年材料界几乎没几个商业级突破（锂电算一个）。这就是 AI 想切入的地方：用模型在虚拟空间里筛掉绝大多数不靠谱的候选。
  • 晶体结构 / 稳定性：无机材料多是周期性排列的晶体。AI 要预测一个假想结构是否「热力学稳定」（不会自己塌掉分解）。DeepMind 的 GNoME 一口气预测了 220 万个晶体结构、其中 38 万个号称最稳定 🟡——但「在绝对零度计算稳定」和「真实实验室能造出来且有用」是两回事（见⑥）。
  • 生成模型 vs 预测模型（类比 LLM）：把 LLM 的「文生图/文生文」搬到材料：预测引擎（如 MatterSim，类似 forward model）给定结构算性能；生成引擎（如 MatterGen，扩散模型，类似 Stable Diffusion）反过来「给定目标性能 → 直接生成符合要求的结构」，即「逆向设计 / inverse design」。开物纪自述的「Prophet 预测引擎 + Creator 生成引擎」双引擎就是这两类 🔴。
  • MLIP（机器学习原子间势）/ 力场：传统算材料性能要跑 DFT（密度泛函理论），又慢又贵。MLIP 是「用神经网络拟合原子间作用力」的代理模型，比 DFT 快几个数量级，是 MatterSim、DeepMind、深势科技等的核心技术之一。可类比 LLM 里「用小模型蒸馏/近似昂贵计算」。
  • 自驱实验室 / self-driving lab（A-Lab）：把「合成 → 表征 → 反馈」交给机器人 + AI 自动闭环跑，不靠人手。这是把「虚拟预测」落到「真东西」的关键环节，也是最难自动化的部分（固态合成不像药物那样靠液体移液就能流水线化）。
  • 从克级到吨级（scale-up）：材料创业的「生死线」。实验室造出 1 克和工厂稳定产 1 吨，是完全不同的工程问题（纯度、良率、成本、设备）。很多 AI 材料公司死在这一步，所以业内常说材料创业「九死一生」。
  • Flagship Pioneering 是谁、为什么被对标：美国一家「风险创造（venture creation）」机构，自己养科学家、自己孵化公司、自己持有平台 IP，最著名的产物是 mRNA 疫苗公司 Moderna 🟡。它不是普通 VC（投别人），而是「自己造资产」。开物纪公开说自己对标的不是陶氏/巴斯夫（卖材料的化工巨头），而是 Flagship 这种「从发现到商业化全链条、自己拥有 IP」的模式 🟢——这正是「自己挖金」的含义。
  • AI4S（AI for Science）：用 AI 加速科学发现的大方向，材料只是其中一支（还有药物、蛋白、催化等）。中国这条线的两个头部是晶泰科技 XtalPi（已港股上市）和深势科技 DP Technology。

①赛道地图 + 早期度判断

category：AI for Science · 材料发现（AI4Materials）。它解决的需求是把「新材料从想出来到能量产」的周期与成本砍掉一个数量级——传统路径靠科学家凭经验试错、动辄十年。开物纪自我定位是「模型—实验—量产」全栈闭环的材料创新平台 🟡。下面按四象限分群（这是判断「威胁从哪来」的关键，后两类最容易被忽视）。

玩家分群（四象限别漏）

  • 现有巨头（化工 / 材料 incumbents）：BASF、Dow、宁德时代等——拥有量产、客户、渠道，但 AI 发现能力弱。它们是开物纪的潜在客户/合作方，也是「自己挖金」时要正面竞争产业化的对手。🟡
  • 垂直创业公司（直接竞品）：海外 Periodic Labs（前 OpenAI/DeepMind 团队，$300M 种子）🟡、Lila Sciences（数亿美元、自动实验室）、Radical AI（A-Lab 负责人 Gerbrand Ceder 加盟）、Orbital Industries（前 Orbital Materials，$50M B 轮，做数据中心材料）、Citrine Informatics、Altrove；中国 晶泰科技 XtalPi、深势科技 DP Technology。开物纪自己就属于这一格。
  • 大厂顺手做（最大威胁之一）：Microsoft（MatterGen/MatterSim/DiG，开物纪创始团队的「母体」）、Google DeepMind（GNoME，220 万材料）、Meta（开放催化剂等）。它们把「预测/生成」做成开源公共品，直接压低创业公司「我有更好模型」的叙事价值。🟡
  • 开源 / 免费替代（易被忽视）：MatterGen 已开源、GNoME 数据库公开、Materials Project、OMat24 等开放数据/模型——意味着「会用 AI 预测材料」本身不再是壁垒，壁垒被迫往「实验闭环 + 量产工程」迁移。🟡
早期度判断 emerging（新兴） （nascent 萌芽 / emerging 新兴 / crowded 拥挤 / mature 成熟）

判断信号： ① 玩家数与新增速度——2025 年集中冒出一批（Periodic、Lila、Radical、开物纪），新进者多但总量仍是「几十家级」，远未到拥挤； ② 融资热度——极热且早期化（Periodic $300M 种子、开物纪成立 3 个月即亿元天使），典型 emerging 的资本前置； ③ 巨头已进场但未独占——MS/Google 做研究与开源，不做量产，留出「把发现落到产品」的窗口； ④ 开源/免费替代已存在（MatterGen/GNoME）——这是把赛道从「nascent」推向「emerging」、并把价值从预测端挤向工程端的关键信号； ⑤ 客户认知未成型、无公认 benchmark 商业标准——没人证明「AI 发现的材料真量产赚钱了」，整条线产品实效全部未验证。

结论：赛道 emerging，开物纪本身处于赛道最早期（成立不到一年、产品未验证）。

⑥国际对标 / Benchmarks（它学谁、差在哪、有什么中国特色）

开物纪是中国公司，这节反过来看它对标的海外是谁、借了什么、差距与本土变量在哪。

对标谁、学了什么

  • 技术母体：Microsoft（MatterGen/MatterSim/DiG）。创始团队直接出自微软研究院材料组，技术栈（扩散生成 + MLIP + scaling）与微软一脉相承——这是它最硬的背书，也是最尴尬之处：它的「武器」微软已经开源了，等于把自己最初的差异化变成行业公共品。
  • 商业模式：Flagship Pioneering（venture creation）。学的是「自己养科学家、自己孵化、持有平台 IP、做到商业化」——把材料发现当「造资产」而非「卖工具」。这是它区别于 Citrine（卖 SaaS）、也区别于 XtalPi/深势（平台 + CRO + SaaS 偏卖铲）的根本叙事 🟢。
  • 实验闭环：Periodic Labs / Lila / Radical AI / A-Lab。学的是「自驱实验室 + 自有实验数据飞轮」——大家都意识到，模型摊平后，真正的稀缺品是高质量实验数据与能跑通的物理闭环。

差在哪（诚实）

  • 弹药量级差一档：开物纪两轮合计「亿元 + 数亿元」人民币；Periodic 一把 $300M 美元种子。海外头部的资本与算力厚度明显更高。🟡
  • 实验/量产基建仍「在建」：对标方多已有运行中的（半）自动实验室，开物纪的实验室/高通量/公斤级平台公开口径是「启动建设」，验证尚未跑出。🔴
  • 「发现新颖性」整条线都被质疑：DeepMind GNoME 的 220 万材料被 UCSB 学者批「几乎没有新颖性/可信度/实用性的三位一体」，且绝对零度计算 ≠ 真实可造——这盆冷水浇在所有「我预测了很多新材料」的叙事上，开物纪不能例外。🟡

中国特色（潜在优势 / 变量）

  • 产业链与工程化更近：中国在电池、电子、化工的量产产业链密集，对一家想「自己挖金、做到量产」的公司是天然主场——从克级到吨级的工程伙伴、设备、产能更易就近找到。这恰好对冲了「实验室到量产」的最大难点。
  • 国产替代 + 政策顺风：新能源、半导体、超导等是国家战略卡脖子方向，AI4S 是政策鼓励赛道，资本（IDG、高瓴、百度、Monolith 等一线机构齐聚）愿意为深科技长周期下注。
  • 人才回流：创始团队是「海外大厂顶尖 research + 产业老兵」的组合，这类「MSR 出走创业」叙事在中国一级市场极受追捧。
  • 差距侧的本土风险：先进实验设备/高端表征仪器、最前沿算力获取仍可能受限；且「自己挖金」要真量产卖钱，要直面成熟化工巨头的成本与渠道。

⑦创业 idea 借鉴

Whitespace / 机会 「单一高价值材料 × 端到端自做量产」这一格仍稀疏。多数玩家要么做通用平台/工具（拥挤、被开源压价），要么宏大叙事追超导/室温超导（周期极长）。Orbital「只打数据中心散热/吸碳材料、且自己产线交付」的窄而深打法是范例——选一个需求确定、客户付费意愿强、不必等十年的材料品类（如 AI 数据中心散热/导热材料、储能特定正极、特定半导体前驱体），把 AI 发现 + 自动实验 + 小批量量产端到端做透，比再造一个「材料界 OpenAI」更可落地。开物纪把「具身散热」列入方向，正卡在这个 why-now 上。

现有玩家共同短板（unmet needs）

① 预测≠可造≠有用——全行业都强于「生成候选」、弱于「真造出来且稳定」，谁能把合成成功率 / 表征自动化 / 数据回流做成飞轮，谁就握住稀缺品。② 缺商业级证据——至今没人能展示「AI 发现的材料量产并卖出钱」的完整案例，第一个跑通的会重定义赛道叙事。③ 从克到吨的工程化几乎是公认黑洞，懂量产的产业老兵稀缺（开物纪专门设「产业 CTO」正是看到这点）。

Why now（为什么现在值得做）

① 生成模型 + MLIP + 自驱实验室三件套同时成熟，且 MatterGen/GNoME 已开源——「会做预测」的门槛被巨头打到地板，反而让小团队能站在巨人肩上、把精力全押在「实验闭环 + 量产」这个真护城河上；② AI 数据中心爆发带来散热/导热/电力材料的硬性新需求（Orbital 已验证可付费）；③ 中国国产替代 + AI4S 政策红利 + 一线资本愿为长周期 deep-tech 前置下注。

可借鉴的打法 + 相邻机会

  • borrowable：(a)「不卖铲子、自己挖金」——当你的模型/能力开始稀缺时，自己持有终端 IP/产品比当工具供应商捕获的价值高得多（前提是你能承受重资产）；(b)「双轨」——一边做能快变现的项目养现金流、一边做 show-muscle 的灯塔项目立叙事；(c)「科学家 + 产业老兵」配队，用产业 CTO 补「实验室到量产」的硬伤；(d) 借「对标 Flagship」一句话把模式叙事讲清楚，便于早期融资。
  • adjacent：(a) 反向做「卖铲子」——给所有「挖金」公司提供自动化高通量实验硬件 / 表征即服务 / 材料数据标注与回流平台（卖水者，不必押注单一材料）；(b) 垂直到单一产业（电池配方优化即服务、半导体前驱体定制）；(c) 做「材料界的 evals」——公认的「可造性 / 可量产性」benchmark 与第三方验证，正是当前最缺的标准。
风险（诚实给） ① 量产九死一生——实验室到吨级是材料创业的生死线，多数死在这；② 开源压低预测价值——MatterGen/GNoME 让「我有更好模型」叙事贬值，逼你必须在工程/数据/量产端建壁垒；③ 资本密集 + 长周期——重资产实验室与产线烧钱、回报以年计，对融资节奏要求极高；④ 巨头与 incumbents 双向夹击——上有 MS/DeepMind 摊平模型、下有 BASF/宁德等掌握量产渠道；⑤ 新颖性/实效未证——整条赛道尚无「AI 材料量产赚钱」铁证，开物纪自己产品实效全部未验证，当前可核实的只有团队 + 方向 + 融资。

⑧现实判断

胜算：开物纪是中国 AI4Materials 里血统最硬、叙事最清晰的早期玩家之一——前微软 MatterGen/MatterSim 核心 + 产业量产老兵 + 一线资本（IDG/高瓴/百度/Monolith）连续两轮加持，且踩中「国产替代 + AI4S + 数据中心新需求」三重 why-now。它「不卖铲子、自己挖金、对标 Flagship」的定位，理论上能捕获比卖 SaaS 高得多的价值，并把护城河放在最难被开源摊平的「实验闭环 + 量产工程」上。软肋：(1) 它最初的技术差异化（MatterGen 系）已被微软开源、变成公共品；(2) 弹药量级仍小于 Periodic 等海外头部一档；(3) 真正的壁垒（公斤级/吨级量产平台）还在「建设中」，没有任何量产产品或营收被验证；(4) 重资产 + 长周期 + 巨头/incumbents 夹击，是结构性的硬仗。早期度上，赛道 emerging、公司本身极早期——这意味着窗口还在、但「能不能挤进去并跑出来」此刻完全取决于它能否第一个把一条材料真量产卖钱，而这一点目前无人做到、它也未证明。
