从这张真表里能读到什么

  • delta 随行权价递减：$270(价内)=0.67 → $285(平值)≈0.50 → $310(价外)=0.22。完全印证"越价内越像股票、越价外越像废纸"。
  • 越价外越便宜但越像彩票：$310 call 只要 $3.6，但 delta 仅 0.22 ≈ 到期价内概率约 16%（看 chance_of_profit）。便宜有便宜的道理。
  • theta（每日损耗）：平值的 $285 每天 −$0.136，即什么都不发生、光过一天，这张合约就缩水约 $13.6（×100）。
  • 买卖价差：$310 那张 bid/ask = $3.00/$4.20，价差占了近 30%！流动性差的合约一进一出先亏一大截（持仓量 OI 高 ≠ 当下价差小）。

3 · 交互一：期权价格 = 内在价值 + 时间价值

选一个真实行权价，拖动股价和剩余天数，看期权价格这条曲线怎么动。两个关键现象：① 曲线是弯的（这就是 delta 在变 = gamma）；② 时间价值会随到期临近而融化（theta）。

两个预设想让你看到的

  • 🚀 赢家：买在低位、股票大涨 → 期权价值随 delta 放大暴涨。但注意：临近到期时时间价值越来越薄，最后几乎只剩内在价值。
  • 💥 归零：买了价外 call，股票一度涨到接近行权价（期权大涨、让你兴奋），结果 6/25 一根大阴线砸下来，到期时跌回价外 → 期权归零。这就是"账面浮盈不落袋，一个大跌全没了 + theta 一直在抽血"的真实写照。

5 · 把真实数字串成的几条铁律

  1. 期权价 = 内在价值 + 时间价值。内在是硬下限（到期价值），时间价值是"还没到期的可能性溢价"，一定随时间流逝而归零。
  2. theta 每天都在抽血。真实的 $285 call 每天 −$13.6/张，且越接近到期、抽得越快（临期 ATM 损耗最猛）。横盘 = 慢性失血。
  3. delta 决定你跟涨多少，但它自己在变（gamma）。所以"涨 $3 赚多少"只是近似——大涨时赚得比 delta×3 更多，大跌时同理。
  4. 浮盈不是落袋。交互二的崩盘预设里，期权一度大幅盈利，但没卖掉，一根大阴线 + 到期就归零。期权是有"保质期"的资产，和股票最大的不同就在这。
  5. 买卖价差是隐性成本。真实链里价外合约价差能到 20-30%，进出各啃一刀。

给你（量化视角）：期权价格是一个多变量函数 V(S, t, σ, r)，希腊字母就是它在各个方向的偏导。你在交互一里拖股价看到的曲线、拖时间看到的塌陷，本质就是 ∂V/∂S(delta)、∂V/∂t(theta) 在真实数字上的可视化。理解了这个，你就从"赌方向"升级到"理解一个可定价、可对冲的资产"了。
