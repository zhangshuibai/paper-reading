量化投资（Quant）入门：用机器学习的语言重新理解它

2026-06-28 · 写给「懂 LLM/ML、但对金融与量化几乎零基础」的你

TL;DR — 一句话版本

量化投资 = 把「赚钱的想法」写成可被数据检验的程序。 你已经会的那套 ML 方法论几乎能一比一迁移过来：特征工程→找因子(factor)， 训练/验证集→样本内/样本外回测， 过拟合→策略曲线拟合， 标签泄漏→未来函数(look-ahead)。 最大的不同只有一个：数据是时间序列、且会「反过来打你」——市场非平稳、还会因为你的交易而改变。这篇用你熟悉的概念把整张地图讲清楚。

1 · 先给你一张「ML → 量化」翻译表

量化领域 90% 的核心概念，你其实早就在 ML 里见过同构物，只是换了名字。点一下卡片看详细对应关系：

读这张表的关键

把量化研究想成一个特殊的监督学习问题：输入是历史行情/财报/另类数据，输出是「未来一段时间的收益方向或大小」。但它有三个 ML 课堂上不强调、实战却要命的特性：

  • 时间不可打乱：不能随机 shuffle 划分训练/测试，否则等于用未来预测过去（典型泄漏）。
  • 非平稳：数据分布一直在漂移（regime change），昨天有效的特征明天可能反转——比 ML 里的 distribution shift 严重得多。
  • 有反身性：你的预测一旦拿去交易，就会改变市场本身（alpha 会衰减/拥挤），相当于「测试集会因为你的模型而变化」。

2 · 量化到底是什么、不是什么

量化投资（Quantitative Investing）指：用数学模型 + 数据 + 程序化规则来做投资决策，而不是靠「我觉得这家公司前景好」的主观判断。一句话——把直觉变成可回测、可复现、可批量执行的规则。

维度主观投资（discretionary）量化投资（quant）
决策依据人对公司/宏观的判断模型对数据的统计规律
覆盖广度几十只深度跟踪几千只同时扫描（像批处理）
可复现性难，依赖个人强，代码即策略
每笔胜率可能很高常常 51%~55%，靠大数定律累积
失败模式看走眼、情绪化过拟合、数据错误、模型失效

先破三个误解

  • 「量化 = 高频交易(HFT)」❌。HFT（微秒级抢单）只是量化里很小、很贵、拼硬件的一支。绝大多数量化是日频/周频/月频，你在笔记本上就能研究。
  • 「量化 = 用 AI 预测明天涨跌」❌。能稳定预测涨跌方向的模型几乎不存在；真实的 alpha 往往是微弱但稳定的统计优势（比如某因子让你在成千上万次下注里平均多赢一点点）。
  • 「回测赚钱 = 实盘赚钱」❌。这是新手最大的坑。漂亮的回测曲线 90% 是过拟合或数据 bug 的产物。怀疑你自己的回测，是量化第一美德。

3 · 最小金融概念集

不需要金融学位，先掌握这几个量就够开工了。

3.1 收益率（return）—— 你的「标签」

价格本身没法直接比较（贵的股票变动绝对值大），所以一切都用收益率说话：

  • 简单收益率：r_t = P_t / P_{t-1} − 1。今天相对昨天涨跌百分之几。
  • 对数收益率：log(P_t / P_{t-1})。量化里更爱用它，因为可加（多日收益直接相加）、且更接近正态分布——和你在 ML 里偏好 log 空间是同一种动机。
  • 复利：净值 = ∏(1 + r_t)。收益靠连乘累积，所以一次大亏的杀伤力远超同等幅度的盈利（亏 50% 要涨 100% 才回本）。

3.2 波动率与回撤 —— 你的「风险」

  • 波动率(volatility)：收益率的标准差，衡量「颠簸程度」。年化波动 = 日波动 × √252（252 是一年交易日数）。
  • 最大回撤(max drawdown)：从历史最高点跌下来的最大幅度。它衡量「最难受的时候有多难受」，直接决定你能不能拿得住、会不会爆仓。

🎲 交互 1：价格是怎么来的？随机游走 vs 趋势

股价常被建模为带漂移的几何布朗运动：每天涨跌 = 一点点确定性趋势(drift) + 大量随机噪声(vol)。拖动滑块，体会「趋势」和「噪声」如何博弈——你会直观感到：当噪声远大于趋势时，看图找规律基本是自欺欺人。

4 · 风险与收益的度量

有了收益和风险，就能定义量化里最重要的「评分函数」。它们的地位相当于 ML 里的 metric。

指标含义ML 类比经验阈值
年化收益 (CAGR)把累计收益摊成「平均每年涨多少」主指标越高越好（但要配合风险看）
夏普比率 (Sharpe)每单位风险换来多少超额收益 = 收益/波动最像「信噪比」>1 还行，>2 很好，实盘能稳定>1.5 已属优秀
最大回撤 (MaxDD)最坏时从高点跌多少worst-case loss越小越好；-30% 已让多数人睡不着
Calmar年化收益 / 最大回撤收益-风险比>0.5 不错
胜率 / 盈亏比赢的次数占比 / 平均盈利÷平均亏损precision 之类高胜率≠赚钱，要和盈亏比一起看
信息比率 (IR)相对基准的超额收益 / 跟踪误差相对 baseline 的提升稳定性评价主动管理的金标准

📐 交互 2：夏普比率计算器

夏普 = (年化收益 − 无风险利率) / 年化波动。它回答「这点收益，值得承担这么大颠簸吗？」。试试：一个年化 30% 但波动 60% 的策略，夏普其实不如年化 10% 波动 8% 的稳健策略。

5 · Alpha、Beta 与因子

这是量化的世界观核心。一个资产的收益可以被拆解：

资产收益 = α (alpha) + β (beta) × 市场收益 + 噪声

  • Beta（β）：随大盘一起涨跌的部分。买指数基金赚的就是 beta——不需要技巧，承担市场风险即可获得。β=1 表示和大盘同步，β=1.5 表示大盘涨1%你涨1.5%。
  • Alpha（α）：剔除市场影响后，靠本事多赚的超额收益。这才是量化真正追逐的东西，也是最稀缺、最容易消失的东西。可以类比：beta 是「baseline 模型」白送的分，alpha 是「你的模型比 baseline 多出来的那点提升」。
  • 因子（factor）：能系统性解释收益的特征维度——这就是量化版的「feature」。经典因子有：
    • 价值 Value 便宜的股票长期跑赢贵的
    • 动量 Momentum 过去涨得好的短期继续涨
    • 规模 Size 小盘股有额外溢价
    • 质量 Quality 盈利稳健、低负债的公司更好
    • 低波动 Low-Vol 波动小的股票风险调整后反而更优

整个「因子投资」流派（Fama-French 三因子/五因子模型）本质上就是一个线性回归：用这些因子去解释一篮子股票的收益。你对线性模型的所有直觉都能用上。

6 · 常见策略大类

策略核心假设一句话
趋势/动量 Momentum涨的会继续涨追随趋势，截断亏损、让利润奔跑
均值回归 Mean-Reversion价格偏离会回归跌多了买、涨多了卖
价值/基本面价格终将反映价值买便宜的好公司
统计套利 Stat-Arb相关资产价差会收敛配对交易：做多A做空B赚价差
套利/做市同一东西不该有两个价赚无风险或低风险价差/价差报价
机器学习选股多因子非线性组合有预测力把上百因子喂给模型排序选股

给 ML 人的提醒：你可能第一反应就是「上 XGBoost / Transformer 预测收益」。可以，但要知道——金融数据信噪比极低（信号也许只有 1%~2%，其余全是噪声），样本量按「独立的市场周期」算其实少得可怜。所以这个领域里，简单稳健 > 复杂炫技，特征的经济学逻辑往往比模型结构更重要。先把线性因子玩透，再上 ML 不迟。

7 · 量化研究流水线 ≈ ML Pipeline

整条工作流和你做一个 ML 项目几乎同构，对照着看：

①
数据
≈ 数据采集/清洗
②
因子/信号
≈ 特征工程
③
回测
≈ 离线评估
④
组合构建
≈ 多输出聚合
⑤
风控
≈ 约束/正则
⑥
实盘
≈ 部署/上线
  1. 数据：行情(价/量)、基本面(财报)、另类数据(新闻、卫星图、舆情)。脏数据是头号杀手——拆股、分红、退市、幸存者偏差都会污染结论。
  2. 因子/信号：把原始数据加工成「对未来收益有预测力的数字」。一个信号通常是横截面排序（今天这 3000 只股票，按某指标从好到坏排队）。
  3. 回测：在历史上模拟「如果当时按这个规则交易，结果会怎样」。下一节专门讲它的陷阱。
  4. 组合构建：把多个信号、多只标的合成一个真实可持有的仓位组合（涉及权重优化、相关性）。
  5. 风控：仓位上限、行业中性、止损、杠杆控制——相当于给优化目标加约束，防止模型在某个方向上裸奔。
  6. 实盘：考虑交易成本、滑点、容量（策略管多少钱就失效）、延迟。很多回测里赚钱的策略，一加上真实成本就归零。

8 · 回测与三大致命陷阱

这是全篇最重要的一节。回测是量化的「离线评估」，但它比 ML 评估危险得多，因为你只有一段历史，反复在上面调参，等于无数次偷看测试集。三大陷阱：

① 未来函数 / 前视偏差（look-ahead bias）= 标签泄漏

用了「当时还不可能知道」的信息。最经典的形式：用今天的收盘价决定今天的买入——可收盘价要等收盘才知道，你根本来不及在那个价位买。防法：信号一律滞后一期生效（今天的信号，最早明天才能交易）。这一行代码 position = signal.shift(1) 漏掉，整个回测就是骗自己。

② 过拟合 / 曲线拟合（overfitting）= 多重检验

你试了 1000 组参数，挑出回测最漂亮的那组——它大概率只是恰好拟合了历史噪声，实盘立刻失效。这和 ML 过拟合同源，但更隐蔽，因为「试了多少次」往往没人记录。防法：样本外测试、walk-forward、参数敏感性分析、限制自由度、对「太完美」的曲线保持警惕。

③ 幸存者偏差（survivorship bias）= 采样偏差

只用「今天还活着的股票」做回测，自动剔除了暴雷退市的——好比只统计考上的学生来证明某学习法有效。结果系统性偏乐观。防法：用包含已退市标的的「point-in-time」数据集。

9 · 需要补的数学/统计

好消息：以你的 ML 底子，几乎不用额外补课。真正高频用到的是：

  • 概率统计：分布、均值/方差、相关性、假设检验、p-hacking 的危害（多重检验）。最重要
  • 线性代数 & 回归：因子模型、协方差矩阵、PCA 降维——全是你熟的。
  • 时间序列：平稳性、自相关、ARIMA/GARCH（波动率建模）。这是相对新的一块，值得专门学。
  • 最优化：投资组合优化（均值-方差/Markowitz）本质是带约束的二次规划。
  • 随机过程（了解即可）：布朗运动、伊藤引理——做衍生品定价才深入需要，选股策略入门用不上。

11 · 给你的学习路线图

  1. 第 1 周 · 打通数据→可视化：用 yfinance 拉 SPY/AAPL，算日收益、累计净值、年化波动，画出来。目标：手不生。
  2. 第 2 周 · 第一个完整回测：手写均线交叉策略（就是交互3），务必加上 shift(1) 和交易成本，算夏普/回撤。目标：理解回测每一步。
  3. 第 3–4 周 · 因子初步：实现一个动量因子或价值因子，对一篮子股票做横截面排序，分 5 组看收益单调性（用 alphalens）。目标：从「单股策略」升级到「横截面因子」思维。
  4. 第 2 月 · 严肃对待过拟合：读 López de Prado，学 walk-forward / purged CV，给你之前的策略做样本外检验，亲眼看它失效。目标：建立怀疑精神。
  5. 第 3 月 · 上 ML + 组合：用 LightGBM 把多个因子组合成排序模型；用 PyPortfolioOpt 做组合优化。目标：把你的 ML 武器库接进来。
  6. 持续 · 打比赛/写记录：去 Numerai 或 Kaggle 量化赛练手；把每个策略的假设、结果、教训写成笔记（防止自己骗自己）。
