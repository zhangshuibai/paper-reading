# 校准 / 帖子分类 / 信号窗口

> 来源：公开 repo `yan-labs/serenity-aleabitoreddit`（2026 自评）。
> **所有命中率均为作者/打包者的估算或自报，非实盘审计**；绝对收益数字**完全未经验证**。
> 用途：给观点**加权**与**祛魅**，不是业绩证明。非投资建议。

## 校准带（2026 复核，估算/自报）
| 指标 | 数值 | 来源等级 |
|---|---|---|
| 30 天方向准确率 | ~61%（30/49 dated calls） | 🟡 估算 |
| 严格 ±10% 命中（30 天） | ~41%（20/49） | 🟡 估算 |
| 60 天内 +20% favorable | ~54%（29/54） | 🟡 估算 |
| 成熟 theses 被价格/基本面部分验证 | 65–75% | 🟡 估算 |
| 最强子集（AI 光子/CPO/InP/内存） | 75–85% | 🟡 估算 |
| 自报绝对收益（如 237% @2026-02、4502% YTD @2026-05） | — | 🔴 **未验证，不得当业绩** |

**怎么用校准**：方向准确率约六成、严格涨幅命中约四成——意味着**单个 call 的胜率有限**，
价值在「**框架帮你筛掉烂标的 + 找到被错杀的卡点**」，而非「跟着每条都买」。最强子集才到 75–85%，
且那是**事后**对「成熟 theses」的统计，含**幸存者偏差**（失败的论点往往不再被提起）。

## 帖子类型分类（决定研究权重）
| 类型 | 权重 | 处理 |
|---|---|---|
| **新 bottleneck 论点** | 最高 | 立刻去验证链条层级 + 定价权 + 14 题清单 |
| **反复重申 / 抄底 dip-buy** | 中 | 看是只是情绪维护，还是有新证据 |
| **供应商地图 supplier map** | 高信息、低即时 | 用于建/校正供应链链条图 |
| **victory lap 晒单/晒收益** | **最低** | 幸存者偏差重灾区；**绝不当入场信号** |

## 信号窗口
公开论点 → 市场验证典型滞后 **5–60 个交易日**（通过订单、SEC 申报、媒体报道、机构建仓兑现）。
→ 含义：**别追已抢跑的事件**（原则 11/红旗）；最佳可操作窗口在论点刚出、机构尚未定价的那段。

## 单票评估时的加权逻辑（workflow a 第⑤步）
1. 先定**帖子类型**（victory lap 直接降权）。
2. 套**校准带**：方向 ~61%、严格涨幅 ~41% → 即便看多也按「胜率有限」处理，靠仓位与定义风险控制下行。
3. **回一手**核实当前价格/基本面/摊薄（theses 快照可能已 stale）。
4. 输出 conviction 档位 + **thesis-breaker** + 风险框架。**非投资建议。**
